This is a set of brushes for Pixel Art with Photoshop. It's shared for free, so don't sell them!

The pack includes:

1. The Basic brushes plus a few one for effects:

- Two basic erarsers.
- 1 Pixel brushes, for outlines and 1px strokes. Hold "Shift" for a perfect pixel stroke!
- Round Brushes, for rounded strokes (Photoshop's circular brush is not pixel friendly!).
- Square Brushes, for rectangular shapes.
- Line brushes, for writing and some effects.
- Pixelated Brushes, for writing and brush effects.
- Screen Tones. A pattern set is included, you can change the texture!

2. Pixel Patterns:

- Grids.
- Scanlines.
- Dots & Screen Tones.
- A few patterns.
- 45ยบ Lines.

3. Pixel Perfect Layer Style. Very useful for instant outlining, you can change the colors easily!.

And that's all, Have fun with pixel art!.
